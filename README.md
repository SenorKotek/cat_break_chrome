# Cat Break Chrome Extension

Простое расширение Chrome, которое периодически показывает видео `cat_break.mp4` поверх открытой веб-страницы.

## Что делает расширение

- Подключается ко всем страницам (`<all_urls>`).
- В случайный момент времени от **1 до 5 минут** показывает оверлей с видео.
- Видео берётся из файла `cat_break.mp4`, лежащего рядом с файлами расширения.

## Как запустить

1. Откройте Chrome и перейдите на страницу `chrome://extensions`.
2. Включите переключатель **Developer mode** (Режим разработчика) в правом верхнем углу.
3. Нажмите **Load unpacked** (Загрузить распакованное расширение).
4. Выберите папку проекта `cat_break_chrome`.
5. Откройте любой сайт и оставьте вкладку активной — видео будет появляться с случайным интервалом от 1 до 5 минут.

## Структура

- `manifest.json` — конфигурация расширения (Manifest V3).
- `content.js` — логика показа оверлея и проигрывания видео.
- `cat_break.mp4` — видео для показа.
